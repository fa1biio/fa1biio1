# 🌐 Meu Site em PHP (MVC Simples)

Projeto de site básico desenvolvido em PHP puro, utilizando uma estrutura organizada no padrão MVC simplificado.

---

## 📁 Estrutura do Projeto

meusite/
│
├── index.php
├── config/
│   └── config.php
├── controllers/
│   └── PageController.php
├── views/
│   ├── layout/
│   │   ├── header.php
│   │   └── footer.php
│   ├── home.php
│   ├── sobre.php
│   └── contato.php
└── public/
    └── css/
        └── style.css

---

## 🧠 Arquitetura

O projeto segue uma estrutura MVC simples:

- **Model** → (não implementado neste exemplo)
- **View** → Arquivos dentro da pasta `/views`
- **Controller** → `PageController.php`
- **Front Controller** → `index.php`

---

## ⚙️ Requisitos

- PHP 7.4 ou superior
- Servidor local (XAMPP, WAMP, Laragon ou similar)

---

## 🚀 Como Executar

### 1️⃣ Instale um servidor local

Você pode usar:

- XAMPP
- WAMP
- Laragon

### 2️⃣ Copie a pasta

Coloque a pasta `meusite` dentro do diretório:

- XAMPP → `htdocs`
- WAMP → `www`
